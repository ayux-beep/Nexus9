# Nexus PWA — Progressive Web App

Install Nexus directly on your phone — no app store needed!

## 📁 Files

```
NexusPWA/
├── index.html        ← Main app
├── manifest.json     ← PWA manifest
├── sw.js             ← Service Worker (offline support)
└── icons/            ← App icons (all sizes)
    ├── icon-72x72.png
    ├── icon-96x96.png
    ├── icon-128x128.png
    ├── icon-144x144.png
    ├── icon-152x152.png
    ├── icon-192x192.png
    ├── icon-384x384.png
    ├── icon-512x512.png
    └── icon-maskable.png
```

## 🚀 Deploy Free (3 Options)

### Option 1 — GitHub Pages (Easiest)
1. Create GitHub repo named: `nexus-app`
2. Upload all files (keep structure)
3. Go to repo Settings → Pages → Branch: main → Save
4. Your app: `https://yourusername.github.io/nexus-app`
5. Open on phone → Install prompt appears ✅

### Option 2 — Netlify (Drag & Drop)
1. Go to [netlify.com](https://netlify.com)
2. Drag the `NexusPWA` folder to the deploy area
3. Get instant URL like `nexus-app.netlify.app`
4. Open on phone → Install ✅

### Option 3 — Vercel
1. Go to [vercel.com](https://vercel.com)
2. Import GitHub repo or drag folder
3. Deploy → get URL
4. Open on phone → Install ✅

## 📱 Install on Phone

### Android (Chrome):
1. Open app URL in Chrome
2. Tap **"Add to Home Screen"** banner at bottom
3. Or: Chrome menu (⋮) → "Add to Home Screen"
4. Tap Install → App icon appears on home screen ✅

### iPhone (Safari):
1. Open app URL in Safari
2. Tap Share button (box with arrow)
3. Scroll down → "Add to Home Screen"
4. Tap Add → App icon appears ✅
